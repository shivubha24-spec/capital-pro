🍬 Crush Shiv

Crush Shiv is a colorful and exciting Match-3 puzzle game inspired by classic candy-matching games, featuring an original design, smooth animations, and engaging gameplay.

✨ Features

- 🎮 Match-3 Puzzle Gameplay
- 🌈 Colorful HD Graphics
- 💥 Amazing Combo & Explosion Effects
- 🧩 Hundreds of Challenging Levels
- 🎁 Daily Rewards
- 🎡 Lucky Spin
- 🪙 Coins & 💎 Diamonds
- 🚀 Powerful Boosters
- 🏆 Leaderboard
- 🏅 Achievements
- 👤 User Profile
- ☁️ Firebase Cloud Save
- 🔐 Secure Login (Google, Email & Guest)
- 💎 Premium Membership
- 🛍️ In-Game Shop
- ⚙️ Admin Panel
- 📱 Optimized for Android

🛠️ Tech Stack

- Kotlin
- Jetpack Compose
- Material Design 3
- Firebase Authentication
- Firebase Firestore
- Firebase Storage
- Google Play Billing
- AdMob
- MVVM Architecture

Currently under active development. New features, levels, and improvements are being added regularly.
